# OpenBB Workspace App Template

A template for creating new OpenBB applications with support for multiple frameworks.

## Features

- Modern project structure supporting multiple frameworks
- Docker support for containerized deployment
- CI/CD integration for automated testing and deployment
- Development tools configuration
- Testing setup
- OpenBB SDK integration
- Framework-agnostic architecture

## Requirements

- Python 3.11+
- OpenBB SDK

## Usage

To create a new OpenBB app from this template:

1. Clone this repository
2. Copy the template files to your new project
3. Update the configuration files with your project details
4. Install dependencies and start development

## Development

To contribute to this template:

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a pull request
